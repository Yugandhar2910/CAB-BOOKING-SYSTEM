=> Overview:
The Cab Booking System is a full-stack web application designed to allow users to book cabs, manage rides, and track ride activity efficiently.
It uses Angular for the front-end user interface and Spring Boot for the backend REST APIs.

This system showcases end-to-end CRUD operations, secure API communication, user management, cab/driver management, and booking workflows.
It follows a clean architecture with separation of concerns between UI, service layer, and data layer.
